API publique

PixShare peut mettre à disposition une API permettant l’envoi automatisé et la gestion technique de fichiers via des requêtes HTTP.

L’accès à l’API peut nécessiter une clé API personnelle. Toute clé API reste strictement personnelle et ne doit pas être exposée publiquement.

Les actions disponibles peuvent inclure notamment :

  • l’envoi de fichiers
  • la génération de liens d’accès ou d’affichage
  • la récupération d’informations techniques liées au fichier
  • la suppression d’un fichier via un lien ou un mécanisme prévu à cet effet

PixShare peut limiter, suspendre ou bloquer l’accès à l’API en cas d’utilisation abusive, de dépassement de quota, de tentative de contournement des limitations ou de non-respect des conditions d’utilisation.
