1. Présentation du service

PixShare est un service permettant d’envoyer et de partager des fichiers via un lien temporaire. Le service est conçu pour un usage simple et rapide sans création de compte utilisateur.

Le service peut être utilisé depuis l’interface web publique ainsi que, lorsque cette fonctionnalité est proposée, via une API permettant l’envoi automatisé de fichiers.

L’utilisation du service implique l’acceptation pleine et entière des présentes conditions.

2. Fonctionnement du service

Lors de l’envoi d’un fichier, un lien unique est généré permettant son accès, son affichage ou son téléchargement selon la nature du fichier et les fonctionnalités disponibles.

  • Les fichiers peuvent être conservés pendant une durée limitée.
  • La durée maximale dépend de la configuration du service.
  • Une fois expiré, le fichier est supprimé automatiquement.
  • L’utilisateur peut également supprimer lui-même les fichiers qu’il a envoyés via l’interface du service ou via les liens fournis lorsque cette fonctionnalité est disponible.
  • Certains fichiers peuvent être conservés sans expiration uniquement si cette option est proposée par le service.

PixShare est un service de partage temporaire et ne doit pas être utilisé comme solution de sauvegarde, d’archivage ou d’hébergement garanti à long terme.

3. Limites techniques

  • La taille maximale des fichiers est limitée.
  • Le service peut limiter certains formats ou certains usages.
  • Le nombre d’envois peut être restreint selon la configuration en vigueur.
  • Le service peut être temporairement indisponible.
  • Les fichiers peuvent être supprimés lors d’une maintenance, d’un redémarrage serveur, d’une erreur technique ou d’une opération de sécurité.

PixShare ne garantit pas la disponibilité permanente des fichiers, ni l’accès continu au service, ni la conservation de tout contenu envoyé.

Tout contenu jugé illégal, dangereux ou abusif pourra être supprimé immédiatement, sans préavis.

Certaines informations techniques minimales peuvent être enregistrées (par exemple adresse IP, informations liées aux requêtes, journaux techniques ou événements d’usage) uniquement pour la sécurité, la stabilité du service et la prévention des abus.

9. API publique

PixShare peut proposer une API publique permettant l’envoi automatisé de fichiers ainsi que, selon les fonctionnalités disponibles, l’accès à certaines informations ou actions liées à ces fichiers.

L’utilisation de l’API nécessite une clé API personnelle lorsqu’elle est exigée par le service. Toute requête effectuée via l’API doit respecter la documentation, les limitations techniques et les présentes CGU.

  • Chaque requête doit inclure une authentification valide lorsque cela est requis.
  • L’API peut permettre l’envoi de fichiers, la récupération de liens d’accès, l’affichage direct de fichiers, ainsi que la suppression d’un fichier lorsque cette action est autorisée.
  • Les liens générés (affichage, téléchargement, suppression) sont strictement liés au fichier concerné.
  • Un lien de suppression ou une URL technique sensible ne doit pas être diffusé publiquement sans nécessité.

Toute utilisation de l’API implique l’acceptation pleine et entière des présentes conditions, au même titre que l’utilisation du site web.

PixShare se réserve le droit, à tout moment et sans préavis, de :

Les erreurs API, y compris notamment celles liées à l’authentification, à l’absence de clé API, à l’épuisement d’un quota ou à une requête invalide, peuvent être renvoyées sous forme de réponses JSON.
